To Execute CoinChange_IO.py:
	1. Make sure it is executable by:
		> chmod +x CoinChange_IO.py
	2. Make sure input text file is in same directory
	3. Execute CoinChange_IO.py by:
		> ./CoinChange_IO.py <[inputfile.txt]>

On completion, results will be written to new text file “[inputfile]change.txt”
	to view:
	 	> cat [inputfile]change.txt
