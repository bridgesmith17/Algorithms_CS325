To Execute tsp.py:
	1. Make sure it is executable by:
		> chmod +x tsp.py
	2. Make sure input text file is in same directory.
	3. Execute CoinChange_IO.py by:
		> ./tsp.py <[inputfile.txt]>

On completion, results will be written to new text file “[inputfile].txt.tour”
	to view:
	 	> cat [inputfile].txt.tour
